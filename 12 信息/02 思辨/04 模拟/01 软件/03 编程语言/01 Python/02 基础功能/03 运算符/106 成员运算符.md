

# 成员运算符

除了以上的一些运算符之外，Python还支持成员运算符，测试实例中包含了一系列的成员，包括字符串，列表或元组。




| 运算符 | 说明 | 举例 |
| ------ | ------------------------------------------------------- | ------------------------------------------------ |
| in     | 如果在指定的序列中找到值返回 True，否则返回 False。     | x 在 y序列中 , 如果 x 在 y 序列中返回 True。     |
| not in | 如果在指定的序列中没有找到值返回 True，否则返回 False。 | x 不在 y序列中 , 如果 x 不在 y 序列中返回 True。 |

举例：


```py
# !/usr/bin/Python
# -*- coding: UTF-8 -*-

a = 10
b = 20
list = [1, 2, 3, 4, 5]

print(a in list)
print(b not in list)

# 修改变量 a 的值
a = 2
print(a in list)
```

输出：

```
False
True
True
```

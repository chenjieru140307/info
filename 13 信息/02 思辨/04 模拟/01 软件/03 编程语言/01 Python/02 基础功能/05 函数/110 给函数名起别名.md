
# 给函数名起别名

函数名其实就是指向一个函数对象的引用，完全可以把函数名赋给一个变量，相当于给这个函数起了一个“别名”：

```py
>>> a = abs
>>> a(-1)
1
```

说明：

-  变量 a 指向 abs 函数，所以也可以通过 a 调用 abs 函数
